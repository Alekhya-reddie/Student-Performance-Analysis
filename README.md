# Student Performance Analysis System

This is a beginner-friendly Java project that analyzes student academic performance.

## Features
- Accepts student name and subject marks
- Calculates total and average
- Assigns grades
- Uses object-oriented programming

## Technologies Used
- Java 17 (Amazon Corretto)
- Visual Studio Code

## How to Run
1. Open the project in VS Code
2. Run `Main.java`
3. Enter student details when prompted

## Future Enhancements
- File storage
- Data visualization
